1. Linux commands / Shell scripts:

	- Shell script control statements, loops, variables, arrays, command line arguments, enabling flags for executing a shell script
	- basic linux commands for file operations
	- linux networking commands such as: telnet, nc (netcat), ping, tcpdump (advanced) etc
	- process commands such as: top, ps, kill, pkill, nohup, lsof etc
	- other miscellaneous commands like: sed, awk(advanced), grep, find, diff, ln, sysctl, systemctl etc
